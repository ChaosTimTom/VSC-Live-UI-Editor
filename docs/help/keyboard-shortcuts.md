# Keyboard Shortcuts & Gestures

Use these when the UI feels “hard to select” or you want precise control.

## Basic actions

- **Click**: select an element
- **Double-click**: start inline text editing (when supported)

## Inline text edit

- `Enter`: commit (save) the text edit
- `Shift+Enter`: insert a newline
- `Escape`: cancel the edit

Tip: if double-click doesn’t start editing, click once to select first, then double-click.

## Layout nudging (small moves)

- Arrow keys: move by 1px
- `Shift` + arrow keys: move by 10px

## Selection modifiers (when you keep selecting the wrong thing)

- Hold `Alt`: prefer the smallest/closest (leaf) element
- Hold `Shift`: prefer a larger/group element
