# Keyboard Shortcuts & Gestures

Use these when the UI feels hard to select or you want precise control.

## Selection

| Shortcut | Action |
|---|---|
| **Click** | Select an element |
| **Double-click** | Start inline text editing (when supported) |
| Hold `Alt` + click | Prefer the smallest / closest (leaf) element |
| Hold `Shift` + click | Prefer a larger / group element |
| `Ctrl+Shift+Click` | Toggle multi-select (add / remove element) |
| `Ctrl+A` | Select all sibling elements of the current selection |
| `Escape` | Clear multi-select; press again to deselect entirely |

> **Tip:** If double-click doesn't start editing, click once to select first, then double-click.

## Inline text edit

| Shortcut | Action |
|---|---|
| `Enter` | Commit (save) the text edit |
| `Shift+Enter` | Insert a newline |
| `Escape` | Cancel the edit |

## Layout nudging (small moves)

| Shortcut | Action |
|---|---|
| Arrow keys | Move by 1 px |
| `Shift` + arrow keys | Move by 10 px |

## Element operations (CRUD)

| Shortcut | Action |
|---|---|
| `Ctrl+D` | Duplicate the selected element |
| `Ctrl+G` | Wrap the selected element in a `<div>` container |
| `Delete` / `Backspace` | Delete the selected element(s) |

These actions also appear as toolbar buttons with shortcut badges when an element is selected.

## Undo / Redo

| Shortcut | Action |
|---|---|
| `Ctrl+Z` | Undo the last change |
| `Ctrl+Shift+Z` | Redo the last undone change |

Undo/redo tracks style changes, moves, text edits, and CRUD operations.

> **Mac users:** Substitute `Cmd` for `Ctrl` in all shortcuts above.
